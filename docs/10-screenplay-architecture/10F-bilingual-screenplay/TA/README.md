# தமிழ் திரைக்கதைப் பதிப்பு

இந்த அடைவில், ஆங்கில முதன்மைத் திரைக்கதையுடன் காட்சி-காட்சியாக ஒத்திசைக்கப்பட்ட தமிழ் நடிப்புப் பதிப்பு வைக்கப்படுகிறது.

|---|---:|---|

| 04 — பாத்திரமும் பசியுற்ற நகரமும் | 18–34 | வரைவு 0.1 முடிந்தது |

**மொத்தம்:** 10 தொடர்காட்சிகள், 72 காட்சிகள்.

## ஒத்திசைவு விதிகள்

- காட்சி எண்கள் ஆங்கிலப் பதிப்புடன் அப்படியே பொருந்த வேண்டும்.
- `TRACE` குறிப்புகள் மாற்றப்படக் கூடாது.
- வசனங்கள் இயல்பான, நடிக்கத்தக்க தமிழில் இருக்க வேண்டும்.
- இது மூலக் காப்பியத்தின் சொற்சொற் மொழிபெயர்ப்பு அல்ல.
- நினைவு, காரணம், சம்மதம், பொறுப்பு ஆகிய வேறுபாடுகள் மாற்றப்படக் கூடாது.

## அடுத்த ஆய்வுநிலை

வரைவு 0.1 கட்டமைப்பில் முடிந்துள்ளது. இது இன்னும் இறுதி வசனப் பதிப்போ படப்பிடிப்புத் திரைக்கதையோ அல்ல. அடுத்த கட்டத்தில்:

- காட்சி மற்றும் `TRACE` ஒத்திசைவு தானியக்கச் சோதனை;
- இயல்பான பேசுதமிழ் ஆய்வு;
- இலக்கிய, பௌத்த மற்றும் தத்துவச் சொல்லாக்க ஆய்வு;
- கதாபாத்திரக் குரல் வேறுபாடு;
- உரக்க வாசிக்கும் நேர அளவீடு;
- ஆங்கில–தமிழ் நாடகத் துடிப்பு ஒப்பீடு

செய்யப்பட வேண்டும்.
