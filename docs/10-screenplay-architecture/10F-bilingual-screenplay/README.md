# 10F — Bilingual Screenplay Edition

## Purpose

This phase maintains a paired English–Tamil screenplay for the *Manimekalai* cinematic adaptation.

- **English master:** `../10E-screenplay-draft/`
- **Tamil performance adaptation:** `TA/`
- **Translation and alignment rules:** `translation_guidelines.md`

The English screenplay remains the structural authority for scene order, unit IDs, primary `SC-*` mappings and absorbed-scene metadata. The Tamil files are performance-oriented adaptations aligned one-to-one with the English master.

## Alignment contract

Every Tamil sequence must preserve:

1. the same sequence number;
2. the same scene numbers;
3. the same `FU-*` or `BR-*` unit ID;
4. the same primary and absorbed `SC-*` IDs;
5. the same dramatic action and ethical outcome;
6. the same distinction between memory, causation, consent and responsibility.

Dialogue may be reshaped for natural Tamil rhythm, but it must not alter character agency or factual causation.

## Review gates

A Tamil sequence is not dialogue-locked until it passes:

- scene and trace parity review;
- natural spoken-Tamil review;
- literary/classical terminology review;
- Buddhist and philosophical terminology review;
- character-voice differentiation review;
- read-aloud timing review.

This is a working bilingual screenplay, not yet a shooting script or final dubbed/subtitled edition.
