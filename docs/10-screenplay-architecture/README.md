   Expand each approved sequence into scene-level beats, reversals, knowledge changes and emotional turns.

   Write the first source-traceable feature screenplay only after the architecture and treatment pass continuity review.

## Architecture rules

- The feature remains Manimekalai's story even when branch narratives occupy the screen.
- Every branch story must alter a present decision, relationship or institution.
- The poem's chapter order may be compressed or intercut, but knowledge order and causal order cannot be falsified.
