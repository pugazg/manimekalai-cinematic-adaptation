## Governance principles

1. A review recommendation is assessed against existing work before becoming backlog.
2. Praise is not validation evidence.
3. Completion claims require an executed check or a cited decision record.
4. Source access is not publication permission.
5. Specialist disagreement and unresolved terminology remain visible.
6. Rejected and deferred decisions remain in the audit trail.
7. Repository discoverability must not overstate screenplay readiness.
8. Licensing must distinguish original repository work from third-party texts and images.
