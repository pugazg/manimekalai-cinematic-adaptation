# Contributing

## Research claims

Each non-trivial claim must carry an evidence label and, where possible, an evidence ID.

## Evidence IDs

Use `EV-0001`, `EV-0002`, and so on. IDs are permanent and must never be reassigned.

## Adaptation decisions

Use `AD-0001`, `AD-0002`, and so on. Each decision must record:

- source observation;
- uncertainty or risk;
- chosen treatment;
- rationale;
- evidence IDs;
- review status.

## Historical uncertainty

Do not convert a plausible reconstruction into a factual assertion. Use `[INTERPRETATION]` and document alternatives.

## Religious terminology

Do not treat later labels, institutions or iconography as automatically applicable to the early Tamil context. This is especially important for the Indra festival.

## File conventions

- Markdown is canonical for research text.
- CSV is canonical for structured registers.
- Lowercase snake_case file names.
- UTF-8 encoding.
